# Client Framework

## Installation

### Cloning

```sh
git clone git@github.com:JamesonRGrieve/ClientFramework.git
```

### Requirements

- NodeJS >= 20
- pnpm (pinned via `packageManager` in `package.json`)

```sh
pnpm install
pnpm dev
```

### Environment

```
APP_NAME=
APP_URI=http://localhost:1109
API_URI=http://localhost:1996
```
